## STEP 1 — Google AdSense 申請（最優先）

### 申請条件（目安）
- コンテンツが一定量ある ✅（4スポット・各ページ動画4本）
- プライバシーポリシーページがある → 要作成（下記参照）
- 独自コンテンツがある ✅

### 手順
1. https://adsense.google.com にアクセス
2. ota@yamaguchi-gp.com で申請
3. サイトURL: `https://ota1014.github.io/doko-real/`
4. 審査通過後、以下のコードを各ページの `<head>` 内に挿入:
   ```html
   <script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-XXXXXXXXXXXXXXXXX" crossorigin="anonymous"></script>
   ```
5. 各ページの `<div class="ad-slot">` を以下に差し替え:
   ```html
   <ins class="adsbygoogle"
        style="display:block"
        data-ad-client="ca-pub-XXXXXXXXXXXXXXXXX"
        data-ad-slot="XXXXXXXXXX"
        data-ad-format="auto"
        data-full-width-responsive="true"></ins>
   <script>(adsbygoogle = window.adsbygoogle || []).push({});</script>
   ```

### 収益目安
| 月間PV | AdSense収入（目安） |
|---|---|
| 1,000 PV | 約¥500 |
| 5,000 PV | 約¥2,500 |
| 15,000 PV | 約¥7,500 |

### 手順
1. https://www.jalan.net/jw/affiliate/ にアクセス
2. アカウント登録（個人で可）
3. 承認後、各スポットページの `btn-jalan` リンクをアフィリエイトURLに差し替え

### 収益目安
- 宿泊予約1件あたり: ¥200〜¥500（予約額の約2〜3%）
- 月10件予約 → ¥2,000〜¥5,000

## 必須作成: プライバシーポリシーページ

AdSense審査に必要。以下の内容でページを作成:
- Cookie/アクセス解析（Google Analytics）の使用について
- 広告配信（AdSense）について
- 免責事項

作成コマンド: 太田さんから指示があれば `/jigyo:hp-demo` 的に自動生成します。

## SEO対策（スポット追加時のルール）

各スポットページのtitleとdescriptionは必ず以下の形式で:
- title: `{スポット名}｜SNS動画でリアルチェック — ドコリアル`
- description: `{スポット名}の{ジャンル}をSNS動画でリアルチェック。{特徴1〜2点}。YouTube動画{N}本まとめ。`

---

## スポット追加の手順

1. `spots/{slug}/` ディレクトリ作成
2. `spots/numazu-port/index.html` をテンプレートにコピー
3. 動画IDは `YouTube {スポット名} 観光 vlog` で検索して差し替え
4. `index.html` のspot-gridに新カード追加
5. `git add . && git commit && git push`
