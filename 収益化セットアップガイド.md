## SEO対策（スポット追加時のルール）

各スポットページのtitleとdescriptionは必ず以下の形式で：
- title: `{スポット名}｜SNS動画でリアルチェック — ドコリアル`
- description: `{スポット名}の{ジャンル}をSNS動画でリアルにチェック。{特徴1〜2点}。YouTube動画{N}本まとめ。`

構造化データ（TouristAttraction / FAQPage / BreadcrumbList）・OGP画像（スポット固有のYouTubeサムネイル）も新規スポットには同様に付与すること。

---

## スポット追加の手順

1. `spots/{slug}/` ディレクトリ作成
2. 既存スポットページ（例: `spots/numazu-port/index.html`）をテンプレートにコピー
3. 動画IDは `YouTube {スポット名} 観光 vlog` で検索して差し替え
4. 該当エリアの `{area}/index.html` のspot-gridに新カード追加
5. `sitemap.xml` に新URLを追記（`add_region.py` 使用時は自動反映）
6. `git add . && git commit && git push`
